# Сливание AVHDX в VHDX
Автоматически находит ВМ в текущей папке и сливает их AVHDX-диски в базовые VHDX.

# Описание:
    1. Сканирует текущую директорию и подпапки на наличие VHDX/AVHDX.
    2. Группирует диски по ВМ (на основе цепочек родительских дисков).
    3. Для каждой ВМ:
       - выключает её (если запущена);
       - сливает все AVHDX в базовый VHDX;
       - логирует результаты.
# Как использовать:
1. Поместите скрипт в папку, где лежат диски ВМ (или в родительскую папку).
   
Пример структуры:
C:\VMs\
  ├── Win10-VM1\
  │   ├── System.vhdx
  │   └── Diff_20240101.avhdx
  └── Win11-VM2\
      ├── OS.vhdx
      └── Snapshot.avhdx
3. Запустите PowerShell от имени администратора.
4. cd "C:\VMs"
5. Разрешите выполнение скриптов (если нужно):
   Set-ExecutionPolicy RemoteSigned -Scope CurrentUser -Force
6. Запустите скрипт:
.\Merge-Avhdx-AutoDiscover.ps1
